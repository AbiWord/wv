A Word 8 converter Web Gateway

This is a web gateway to mswordview, which is a GPLed converter for word 8 (office 97) documents to html. The standalone program for unix et al is available as well. Dont use this to convert information you wouldnt want me to see, coz if the conversion doesnt work, ill be using the file you convert to try and extend what mswordview can support, which will require me to read it. Also due to possible bugs, the converter will give up if it isnt finished after 4 minutes, so very big docs that would legitimately be converted will be truncated by that time limit.
